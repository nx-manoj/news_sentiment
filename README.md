# 📰 News Sentiment Analysis

A simple Python project that analyzes news headlines and classifies them into **Positive**, **Negative**, or **Neutral** sentiment.

---

## 🚀 Features

* Analyze real-time or static news headlines
* Classifies sentiment using NLP techniques
* Easy to run and modify
* Beginner-friendly project

---

## 🛠️ Tech Stack

* Python 🐍
* NLP (TextBlob / VADER / custom model)
* Matplotlib (for visualization)

---

## 📂 Project Structure

```
news_sentiment/
│── news_sentiment.py
│── data/
│── models/
│── README.md
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/nx-manoj/news_sentiment.git
```

2. Navigate to the project folder:

```
cd news_sentiment
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the script:

```
python news_sentiment.py
```

---

## 📊 Example Output

```
"Stock market crashes due to global crisis" → Negative
"New technology improves healthcare system" → Positive
```
